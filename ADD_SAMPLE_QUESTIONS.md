# إضافة أسئلة تجريبية لقاعدة البيانات

## المشكلة
عدد أسئلة الاهتمامات وأسئلة أسلوب التعلم يظهر كـ 0 لأن جدول `Questions` في قاعدة البيانات فارغ.

## الحل

### الطريقة 1: استخدام ملف SQL

1. افتح ملف `sample_questions.sql`
2. انسخ محتوى الملف
3. نفذ الاستعلام في قاعدة البيانات (باستخدام phpMyAdmin، MySQL Workbench، أو أي أداة إدارة قاعدة بيانات)

أو استخدم terminal:

```bash
mysql -h mysql5047.site4now.net -u abf0c2_umj -p db_abf0c2_umj < sample_questions.sql
```

### الطريقة 2: إضافة أسئلة يدوياً

استخدم استعلام SQL التالي لإضافة أسئلة:

```sql
-- أسئلة الاهتمامات
INSERT INTO Questions (Text, Type, Category) VALUES
('ما هي المجالات الأكاديمية التي تهمك أكثر؟', 'interests', 'أكاديمي'),
('ما هي المواد الدراسية التي تفضل دراستها؟', 'interests', 'أكاديمي'),
('ما هي الاهتمامات العلمية التي ترغب في استكشافها؟', 'interests', 'علمي'),
('ما هي المجالات المهنية التي تثير اهتمامك؟', 'interests', 'مهني'),
('ما هي الأنشطة التي تستمتع بها في وقت الفراغ؟', 'interests', 'شخصي');

-- أسئلة أسلوب التعلم
INSERT INTO Questions (Text, Type, Category) VALUES
('ما هو أسلوب التعلم المفضل لديك؟', 'learning_style', 'تعليمي'),
('كيف تفضل استقبال المعلومات الجديدة؟', 'learning_style', 'تعليمي'),
('هل تفضل التعلم النظري أم العملي؟', 'learning_style', 'تعليمي'),
('ما هي البيئة التعليمية التي تناسبك أكثر؟', 'learning_style', 'تعليمي'),
('كيف تفضل تنظيم وقت التعلم؟', 'learning_style', 'تعليمي');
```

### الطريقة 3: إضافة من واجهة الإدارة

1. افتح صفحة إدارة الاستبيانات
2. اضغط على زر "إضافة سؤال جديد"
3. املأ البيانات:
   - **نوع السؤال**: اختر "الاهتمامات" أو "أسلوب التعلم"
   - **الفئة**: أدخل الفئة (مثل: أكاديمي، تعليمي، إلخ)
   - **السؤال**: اكتب نص السؤال
   - **الخيارات**: أضف الخيارات (اختياري)
4. احفظ السؤال

## التحقق من البيانات

بعد إضافة الأسئلة، تحقق من وجودها:

```sql
SELECT QuestionID, Text, Type, Category FROM Questions ORDER BY Type, QuestionID;
```

أو استخدم Swagger UI:
1. افتح `http://localhost:8000/api-docs`
2. جرب endpoint `GET /api/v1/survey/questions`
3. يجب أن ترى الأسئلة في الاستجابة

## ملاحظات مهمة

### أنواع الأسئلة المدعومة:

1. **interests** (الاهتمامات)
   - يظهر في البطاقة اليمنى
   - يتم حسابها باستخدام: `questions.filter(q => q.type === 'interests')`

2. **learning_style** أو **learning-style** (أسلوب التعلم)
   - يظهر في البطاقة اليسرى
   - يتم حسابها باستخدام: `questions.filter(q => q.type === 'learning-style')`

### ملاحظة على Type:

- الـ API يدعم كلا التنسيقين: `learning_style` و `learning-style`
- في Frontend، يتم تحويل `learning_style` إلى `learning-style` تلقائياً
- في قاعدة البيانات، يمكن استخدام أي من التنسيقين

## بعد إضافة الأسئلة

1. أعد تحميل صفحة إدارة الاستبيانات
2. يجب أن ترى العدد الصحيح:
   - عدد أسئلة الاهتمامات: 5
   - عدد أسئلة أسلوب التعلم: 5
   - إجمالي الأسئلة: 10

3. إذا استمر ظهور 0:
   - تحقق من Browser Console للأخطاء
   - تحقق من API Server Console للـ debug logs
   - تحقق من أن البيانات موجودة في قاعدة البيانات
